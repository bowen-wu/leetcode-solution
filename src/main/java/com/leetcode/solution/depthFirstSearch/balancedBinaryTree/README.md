1. 分治法的 combine，除了查看左右子树是否平衡，还要看当前树是否平衡 => 自顶向下
2. getHeight 会被重复调用

4. getHeight 具有两个功能，一个是计算高度，一个是判断是否平衡，如果不平衡，返回 -1
