## 翻转对

<https://leetcode.cn/problems/reverse-pairs/>

### 思路

1. 归并排序 => 合并两个有序数组 => [start, mid] 和 [mid + 1, end] 是有序的
2. 在 [start, end] 找重要翻转对
3. 不能使用除法，Java 中除法是地板除

### 总结

1. 合并两个有序数组，剩余部分使用 while 循环，而不是 if
2. 合并两个有序数组，剩余部分使用 while 循环，而不是 if
